AI ASSISTANT TOKEN OPTIMIZATION - TASK LIST
============================================
Created: 2026-01-11
Goal: Reduce AI API token usage by 60-70% while preserving full functionality

OVERVIEW:
---------
Currently, the AI assistant returns full scale data (intervals, noteDegrees, chordTones) 
for each recommendation, which uses excessive tokens. We will optimize by:
1. Having AI return only: scaleName, rootNote, rationale, genreContext, difficulty
2. Using GPT-4o-mini's consistent naming to map to our internal scale database
3. Populating intervals, noteDegrees, chordTones from our EXTENDED_SCALE_INTERVALS
